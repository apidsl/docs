
## run [<span style='font-size:20px;'>&#x270D;</span>](https://github.com/apidsl/docs/edit/main/COMMAND/RUN.md)

run file script

run as option param
```bash
apidsl --run "test.apidsl"
```

run as command

```bash
apidsl 'run("test.apidsl")'
```

with logs 
```bash
apidsl -l 'run("test.apidsl")'
```
